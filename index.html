Task

Create a CSS layout for this website.
The CSS file can be linked via the URL.

Whether you choose a 2 or 3 column layout is up to you.
This task does not focus on the graphical representation of the layout, but instead the technical implementation.

The navigation must be styled using the :hover pseudo class, while the active menu point must use the body class.
Moreover, at least the #header title should be replaced by an image using "imagereplacement".

Please make sure that the text can be read easily.

Targetbrowser:
Firefox
Google Chrome
Internet Explorer 6
Internet Explorer 7
Internet Explorer 8
Internet Explorer 9
